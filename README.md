# УчебПлатформа — Дизайн-прототип

Платформа дистанционного обучения — демо-версия без базы данных.

## Запуск

Просто откройте `index.html` в браузере.

## Что работает

- ✅ Авторизация / регистрация
- ✅ Каталог курсов (3 демо-курса)
- ✅ Страница курса с боковой навигацией
- ✅ Модули: Введение → Теория → Видео → Тест → Экзамен
- ✅ Блокировка: следующий модуль только после завершения предыдущего
- ✅ Промежуточные тесты с автопроверкой
- ✅ Экзамен по билетам
- ✅ Профиль пользователя
- ✅ **Конструктор курсов** (как Teachable / Notion):
  - Список курсов — карточки с количеством уроков
  - Sidebar слева — разделы курса (Введение, Теория, Видео, Тест, Экзамен)
  - Редактор справа — большое поле для текста, карточки для видео/вопросов
  - Чекбоксы в сайдбаре — включить/выключить раздел
  - Кнопка «+ Добавить раздел» — добавить новый
  - Кнопка «💾 Сохранить» — в сайдбаре внизу
- ✅ Адаптивный дизайн (мобильные, планшеты, десктоп)
- ✅ Фирменные цвета: серый + оранжевый (#E87A2E)

## Что НЕ работает

- ❌ Нет базы данных (данные хранятся только в памяти, сбрасываются при обновлении)
- ❌ Нет загрузки файлов
- ❌ Нет хостинга
